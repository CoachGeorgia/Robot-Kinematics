# Robot-Kinematics
### Robot Kinematics & Machine Learning 
Robotic Arm software for Kinematics and Learning. 
Code examples modified from DynamixelSDK and easyDyamixel.
### Hardware
Robot Arm: WidowX200
### Firmware 
Dynamixel Communication Protocol 2.0
### Supported OS
Windows
